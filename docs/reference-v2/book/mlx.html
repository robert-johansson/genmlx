GenMLX API Reference

MLX Operations

Thin ClojureScript wrapper over Apple’s MLX framework (@frost-beta/mlx). All operations produce lazy computation graphs – call mx/eval! to materialize. Values stay as MLX arrays from sampling through scoring through gradient computation.

Source: src/genmlx/mlx.cljs, src/genmlx/mlx/random.cljs
