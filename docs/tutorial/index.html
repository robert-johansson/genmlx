Tutorial: Probabilistic Programming with GenMLX

This tutorial introduces probabilistic programming through GenMLX — a GPU-accelerated implementation of the Generative Function Interface in ClojureScript on Apple Silicon. Each chapter builds on the previous one, progressing from basic concepts to advanced inference.

Prerequisites: macOS with Apple Silicon, Bun (or Node.js 18+), and npm install in the GenMLX repo. Basic familiarity with ClojureScript or any Lisp is helpful but not required.

Chapters

  1. Introduction — What is probabilistic programming? Why GenMLX?
  2. Generative Models — Writing models with gen, trace, and distributions
  3. Conditioning — Observations, choicemaps, and generate
  4. Inference Basics — Importance sampling, Metropolis–Hastings
  5. Combinators — Map, Unfold, Switch, Scan, and compositional models
  6. Advanced Inference — HMC, SMC, SMCP3, variational inference
  7. Vectorized Inference — Shape-based batching, vsimulate, vgenerate
  8. Gradients and Learning — Score gradients, ADEV, parameter optimization
  9. Neural Probabilistic Models — nn->gen-fn, amortized inference
  10. Custom Extensions — defdist, custom combinators, GFI contracts

How to use this tutorial

Each chapter contains explanatory text with code examples. To run the examples:

;; Save a snippet to a file, e.g. example.cljs, then run:
bun run --bun nbb example.cljs

Code blocks show the source followed by a dark output block showing what you would see in your terminal. You can copy any code block by hovering and clicking the copy button.

The tutorial assumes you have cloned the GenMLX repo and run npm install. All examples use the GenMLX source directly from src/.
